HTML이란 무엇인가?

HTML is the standard markup language for Web pages.With HTML you can create your own Website. Web pages can be created and modified by using professional HTML editors.However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac).

We believe using a simple text editor is a good way to learn HTML.
